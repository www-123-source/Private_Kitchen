# 购物车功能使用指南

## 概述

Private Kitchen 应用已实现完整的购物车功能，支持商品的添加、修改数量、删除和结算等操作。

## 主要功能

### 1. 购物车管理

- **添加商品**：从菜品详情页点击"添加到购物车"按钮
- **修改数量**：在购物车中点击加减按钮或使用数量选择器
- **删除商品**：点击删除按钮或滑动删除
- **清空购物车**：通过菜单或按钮清空所有商品
- **查看总价**：自动计算并显示总价

### 2. 智能功能

- **库存检查**：自动检测并移除已下架的商品
- **重复商品**：同一商品会合并数量而非重复添加
- **状态同步**：购物车状态实时同步更新

## 使用流程

### 1. 添加商品到购物车

1. 在菜品详情页选择数量
2. 点击"添加到购物车"按钮
3. 成功提示会显示在顶部

### 2. 管理购物车

1. 点击导航栏的购物车图标进入购物车
2. 查看已添加的商品列表
3. 使用以下操作管理商品：
   - 点击 `+` 增加1个数量
   - 点击 `-` 减少1个数量
   - 点击数量文本打开数量选择器
   - 点击垃圾桶图标删除商品

### 3. 结算流程

1. 确认商品信息无误
2. 点击"去结算"按钮
3. 选择支付方式
4. 完成支付
5. 支付成功后自动清空购物车

## 数据模型

### CartItem

```swift
struct CartItem: Identifiable {
    let id = UUID()
    let dish: Dish        // 关联菜品
    var quantity: Int     // 数量
    var addedAt: Date     // 添加时间
}
```

### CartManager

主要方法：
- `addItem()` - 添加商品
- `updateQuantity()` - 更新数量
- `removeItem()` - 移除商品
- `clearCart()` - 清空购物车
- `getCartSummary()` - 获取购物车摘要

## UI 组件

### CartView - 购物车主视图

- 商品列表展示
- 数量调整控件
- 总价计算
- 结算按钮

### CartItemRow - 购物车商品行

- 商品图片
- 商品信息
- 价格和数量
- 操作按钮

### QuantityPickerView - 数量选择器

- 滑轮式数量选择
- 最大限制支持
- 实时预览

## 通知机制

购物车操作会发送通知：
- `cartItemAdded` - 添加商品
- `cartItemRemoved` - 移除商品
- `cartCleared` - 清空购物车
- `cartUpdated` - 更新购物车

## 注意事项

1. **商品下架**：已下架的商品会从购物车自动移除
2. **数量限制**：单个商品最多99个
3. **实时同步**：购物车数据实时同步到数据库
4. **错误处理**：异常操作会有相应提示
