# 私厨 · 应用原型

基于 [2026-02-28 设计文档](../plans/2026-02-28-private-kitchen-design.md) 制作的可交互 Web 原型，用于预览 iOS 应用「私厨」的主要界面与流程。

## 如何查看

1. 用浏览器直接打开 `index.html`（建议 Chrome / Edge）。
2. 或在本项目根目录启动本地服务后访问 `docs/prototype/index.html`。

## 原型范围

| 模块       | 内容说明 |
|------------|----------|
| **启动与身份** | 启动页、选择「厨房管理员」或「家庭成员」进入对应端 |
| **管理员端**   | 竖向侧边栏：菜品、订单、成员、菜谱、设置；各 Tab 下的列表与筛选 |
| **顾客端**     | 竖向侧边栏：点餐、购物车、我的订单、菜谱；点餐网格、购物车、订单列表 |
| **私人菜谱**   | 两端共用：按地区筛选、菜谱卡片（名称、商户、评分、评论数） |
| **交互**       | 身份切换、Tab 切换、菜品卡片点击 → 菜品详情浮层（加购） |

## 设计说明

- **风格**：温暖厨房感，米白/奶油底 + 陶土红主色 + 鼠尾草绿辅助。
- **字体**：Noto Serif SC（标题）+ DM Sans（正文）。
- **设备**：以 375×812 手机框展示，便于与 iOS 设计对照。

## 与设计文档对应关系

- 启动流程：启动页 → 选择身份 → 管理员端 / 顾客端。
- 导航：与文档中「竖向侧边栏」一致（管理员 5 项、顾客 4 项）。
- 数据与状态：列表内容为占位示例（菜品分类、订单状态、成员角色等与文档枚举一致）。

如需导出为图片或补充更多页面，可在原型基础上继续扩展。
